# Pre-Record Setup Checklist

Run through before hitting record. Each item is one decision.

## Audio
- [ ] Mic is the right one (USB / lavalier / built-in — pick best available)
- [ ] Mic input level tested in Loom (peak at -12 to -6 dB during normal speech)
- [ ] No fan / AC / outside noise audible — pause AC if needed
- [ ] Headphones on (prevents echo / playback bleed)

## Screen
- [ ] Resolution 1920×1080 minimum (1440p preferable for code zoom)
- [ ] Browser zoom set so text is readable at thumbnail size (Cmd-+ at least 1–2 times)
- [ ] All notifications muted (macOS Focus mode "Do Not Disturb")
- [ ] All non-essential tabs closed (only what the walkthrough needs)
- [ ] Dock auto-hidden, menu bar items minimal (hide Slack, Discord, etc.)
- [ ] Demo data pre-loaded — don't waste recording time on "wait, let me find that"

## Camera (if face-on)
- [ ] Camera lens clean
- [ ] Light source in front (window or ring light), not behind
- [ ] Eyes at top third of frame
- [ ] Background: neutral, no clutter, no clock visible (clocks date the video)

## Environment
- [ ] Door closed
- [ ] Phone on Do Not Disturb (face down)
- [ ] Water within reach
- [ ] Bathroom break done

## Content
- [ ] Hook rehearsed once out loud
- [ ] Beats memorized to label level (don't read from script)
- [ ] Numbers / facts double-checked against `data/posts.json` or source
- [ ] Close line ready ("What version are you on?", etc.)
- [ ] Knowing when to stop (target duration ± 30%)

## Pre-flight
- [ ] Loom app open + logged in
- [ ] Recording target set (full screen / window / camera-only)
- [ ] Recording quality set to HD
- [ ] Trash test recording done (3 sec of "test test test", confirm audio + video both captured)
- [ ] Trash test deleted
